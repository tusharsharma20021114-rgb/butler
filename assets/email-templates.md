# 📧 Email Templates

## Template 1: Introduction / First Contact

```
Subject: [Your Name] — [Context/Referral/Reason]

Hi [Name],

I'm [Your Name], [one-line description of who you are and why you're reaching out].

I'm writing because [specific reason — be direct].

[1-2 sentences of value proposition — what's in it for them]

Would you be open to [specific ask — a call, a meeting, reviewing something]? I'm flexible on timing.

Best,
[Your Name]
[Title / Contact Info]
```

## Template 2: Project Update / Status Report

```
Subject: [Project Name] Status Update — [Date]

Hi [Name/Team],

Here's a quick update on [Project Name]:

**✅ Completed this week:**
- [Achievement 1]
- [Achievement 2]

**🔄 In progress:**
- [Task 1] — ETA: [Date]
- [Task 2] — ETA: [Date]

**🚧 Blockers:**
- [Blocker 1] — Need: [what you need to unblock]

**📅 Next week's focus:**
- [Priority 1]
- [Priority 2]

Let me know if you have questions or want to discuss anything.

Best,
[Your Name]
```

## Template 3: Thank You / Follow-Up

```
Subject: Thank you — [Context]

Hi [Name],

Thank you for [specific thing — your time, your help, the opportunity, the introduction].

[1-2 sentences about what you took away or what happens next]

I'll [next step — send the document, follow up on Friday, schedule the next meeting].

Thanks again!

Best,
[Your Name]
```

## Template 4: Request for Help / Favor

```
Subject: Quick request — [What you need]

Hi [Name],

I hope you're doing well. I have a [small/quick] request:

[Clearly state what you need — 1-2 sentences max]

**Context:** [Brief background so they understand why]

**What I need from you:** [Specific, actionable ask]

**Timeline:** [When you need it by]

I understand if you're too busy — no pressure at all. Happy to [offer something in return] if helpful.

Thanks,
[Your Name]
```

## Template 5: Apology / Mistake Acknowledgment

```
Subject: Apology regarding [Issue]

Hi [Name],

I want to apologize for [specific mistake — own it clearly].

[What happened — brief, factual, no excuses]

Here's what I'm doing to fix it:
1. [Immediate corrective action]
2. [Prevention measure for the future]

I take full responsibility and will ensure this doesn't happen again. Please let me know if there's anything else I should do.

Sincerely,
[Your Name]
```

## Template 6: Cold Outreach (Professional)

```
Subject: [Specific value prop in 5 words or fewer]

Hi [Name],

I noticed [something specific about them/their work — shows you did your homework].

I'm working on [brief description of your project/company], and I think [specific reason this is relevant to them].

Would you be open to a 15-minute call to [specific agenda]? No worries at all if not — I know you're busy.

[Optional: link to relevant work, demo, or portfolio]

Best,
[Your Name]
```
