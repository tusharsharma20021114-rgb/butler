# Planning Strategies — Butler Reference

## Eisenhower Matrix

Categorize every task into one of four quadrants:

```
                    URGENT              NOT URGENT
              ┌─────────────────┬─────────────────┐
  IMPORTANT   │  🔥 DO FIRST    │  📅 SCHEDULE     │
              │  Do immediately │  Block time for  │
              │  Crisis, hard   │  Deep work,      │
              │  deadlines      │  growth, strategy│
              ├─────────────────┼─────────────────┤
  NOT         │  👤 DELEGATE    │  🗑️ ELIMINATE    │
  IMPORTANT   │  Hand off or    │  Say no, remove, │
              │  batch process  │  stop doing      │
              └─────────────────┴─────────────────┘
```

### How to Apply
1. List all tasks for the day/week.
2. For each task, ask: "Is this **urgent**?" (has a hard deadline or immediate consequence)
3. Then ask: "Is this **important**?" (contributes to long-term goals or values)
4. Place in the corresponding quadrant.
5. Work through quadrants in order: DO → SCHEDULE → DELEGATE → ELIMINATE.

## Time Blocking

### Rules
1. **Morning block (2-3 hours)**: Protect for deep, focused work. No meetings, no email.
2. **Midday block (1-2 hours)**: Meetings, calls, and collaborative work.
3. **Afternoon block (2-3 hours)**: Mixed — lighter tasks, email, admin, and planning.
4. **Buffer blocks (30 min)**: Leave gaps between blocks for unexpected tasks.

### Template
```
09:00 - 11:30  🧠 Deep Work (high-priority task)
11:30 - 11:45  ☕ Break
11:45 - 12:30  📧 Email & Communications
12:30 - 13:30  🍽️ Lunch
13:30 - 14:30  👥 Meetings
14:30 - 16:00  📋 Task Execution
16:00 - 16:15  ☕ Break
16:15 - 17:00  📝 Review & Plan Tomorrow
17:00 - 17:30  🔄 Buffer / Overflow
```

## Pomodoro Technique

1. Choose a task.
2. Set timer for **25 minutes** (one "pomodoro").
3. Work with full focus — no distractions.
4. Take a **5-minute break**.
5. After 4 pomodoros, take a **15-30 minute break**.

### Best For
- Tasks you're procrastinating on
- Deep work that requires sustained focus
- When you need to track how long tasks actually take

## Getting Things Done (GTD) — Simplified

### The 5-Step Workflow
1. **Capture**: Write down everything (tasks, ideas, commitments).
2. **Clarify**: For each item, ask: "Is it actionable?"
   - No → Trash it, file it, or add to "someday/maybe" list
   - Yes → Define the next physical action
3. **Organize**: Put actions in the right context (work, personal, errands, calls).
4. **Reflect**: Weekly review — update lists, check progress, plan ahead.
5. **Engage**: Do the work, choosing based on context, time available, and energy.

### The 2-Minute Rule
If a task takes **less than 2 minutes**, do it immediately. Don't add it to a list.

## Weekly Review Checklist

Every week (recommend: Friday afternoon or Sunday evening):

- [ ] Clear inbox to zero
- [ ] Review calendar for next week
- [ ] Update task list — remove completed, add new
- [ ] Identify top 3 priorities for next week
- [ ] Review "someday/maybe" list
- [ ] Reflect: What went well? What needs improvement?
- [ ] Plan Monday's time blocks

## Energy Management

Match task type to your energy level:

| Energy Level | Task Type | Time of Day (typical) |
|-------------|-----------|----------------------|
| 🔋 High | Deep thinking, writing, complex problems | Morning |
| 🔋 Medium | Meetings, collaboration, creative work | Late morning / early afternoon |
| 🔋 Low | Email, admin, routine tasks | Late afternoon |
| 🪫 Depleted | Break, walk, recharge | After lunch, end of day |
